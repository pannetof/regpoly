# Using the REGPOLY web UI

> **Status**: Phase 0 placeholder. Final content authored in Phase 5–6 against the rewired web app.

```bash
uv run regpoly-web --db packages/regpoly-web/var/regpoly.db
```

The web UI is a FastAPI shell over a SQLite-backed result store. Use it to launch full-period or tempering searches, browse tested generators, and publish to the catalog.
