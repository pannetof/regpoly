# F₂-linear generators

> **Status**: Phase 0 placeholder. Full content authored in Phase 6.

This page is the umbrella reference for F₂-linear pseudo-random number generators as REGPOLY models them. Phase 6 fills in:

1. State, recurrence, and output map over $\mathbb{F}_2$.
2. The characteristic polynomial and the full-period condition.
3. Equidistribution properties: $d(t)$, $\Delta(t_1, \ldots, t_d)$, antithetic structure.
4. Combined generators: state product, output XOR, char-poly product.
5. Tempering as a post-processing linear map.
6. Pointers to the per-family pages under [`../generators/`](../generators/index.md) and to the algorithmic specs in this section.
