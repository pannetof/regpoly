# Python API

Reference for the **public** Python API of REGPOLY across the three
runtime packages (`regpoly`, `regpoly_legacy`, `regpoly_web`). Pages
are generated by Sphinx [`autodoc`](https://www.sphinx-doc.org/en/master/usage/extensions/autodoc.html)
from in-source NumPy-style docstrings, rendered through
[`napoleon`](https://www.sphinx-doc.org/en/master/usage/extensions/napoleon.html).

Every wrapper class that has a C++ counterpart links to it via
`:cpp:class:` cross-references — click the link in a "See Also" or
docstring body to jump to the C++ implementation page.
